## Upgrade Instructions

Follow these steps to update your Hello installation.

<Steps>
  <Step title="Backup Current Setup" icon="database">

    Create a backup of your configuration and database.

    ```
    cp config.json config.json.bak
    pg_dump hello_db > hello_backup.sql
    ```

  </Step>
  <Step title="Update Package" icon="package">

    <CodeGroup tabs="npm,yarn,pnpm">
      ````bash
      npm install hello-sdk@latest
      ````

      ````bash
      yarn add hello-sdk@latest
      ````

      ````bash
      pnpm add hello-sdk@latest
      ````
    </CodeGroup>

  </Step>
  <Step title="Run Migrations" icon="zap">

    Apply database migrations.

    ````bash
    npx hello-sdk migrate
    ````

  </Step>
  <Step title="Restart Services" icon="refresh-cw">

    Restart your application server.

    ```
    pm2 restart hello-app
    ```

  </Step>
</Steps>

<Callout kind="tip">
  Check the [migration guide](/changelog#breaking-changes) for v1.2.0 specifics. Test in staging before production.
</Callout>
